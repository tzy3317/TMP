# Cue-Pro 项目规范 (Specification)

> 本文档定义 Cue-Pro 项目的所有技术约定、命名规范和配置标准。
> 版本：v1.0.0 | 最后更新：2026-05-06

---

## 1. 项目概述

### 1.1 项目信息

| 属性 | 值 |
|------|-----|
| 项目名称 | Cue-Pro |
| 开发语言 | Python |
| LLM 协议 | OpenAI 兼容协议 |
| 配置方式 | 环境变量（`.env` 文件） |

### 1.2 设计原则

1. **单一职责**：每个配置文件只负责一类配置
2. **显式优于隐式**：所有配置必须声明，禁止魔法值
3. **自文档化**：配置项必须附带注释说明用途和示例
4. **安全优先**：敏感信息（API Key 等）严禁提交到代码仓库

---

## 2. 环境变量命名规范

### 2.1 命名规则

- 全部采用 **大写字母 + 下划线** 格式（`UPPER_SNAKE_CASE`）
- 使用 **功能前缀** 归类，避免命名冲突
- 格式：`{功能域}_{具体配置名}`

### 2.2 前缀定义

| 前缀 | 用途 | 示例 |
|------|------|------|
| `LLM_` | LLM 相关配置 | `LLM_BASE_URL` |
| `DB_` | 数据库配置（预留） | `DB_HOST` |
| `APP_` | 应用级配置（预留） | `APP_DEBUG` |

### 2.3 当前生效前缀

本项目当前仅使用 `LLM_` 前缀。

---

## 3. 环境变量分类

### 3.1 必配项（Required）

必须手动填写，否则应用无法启动。

| 变量名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `LLM_BASE_URL` | string | API 基础地址 | `https://api.openai.com/v1` |
| `LLM_MODEL` | string | 模型名称 | `gpt-4o` |
| `LLM_API_KEY` | string | API 密钥 | `sk-xxxxxxxx` |

### 3.2 可选项（Optional）

当前版本所有调参项（temperature、max_tokens 等）均内置于代码默认值中，**不通过环境变量暴露**。

如需覆盖，请直接修改代码中的配置字典。

### 3.3 预留项（Reserved）

以下变量名已预留，供未来版本使用：

| 变量名 | 用途 | 状态 |
|--------|------|------|
| `LLM_EMBEDDING_MODEL` | Embedding 模型名称 | 未启用 |
| `LLM_API_VERSION` | API 版本号（Azure 等需要） | 未启用 |
| `LLM_TIMEOUT` | 请求超时时间（秒） | 未启用 |

## 4. 配置文件规范

### 4.1 文件清单

| 文件名 | 作用 | 是否提交仓库 |
|--------|------|-------------|
| `.env.example` | 环境变量模板 | ✅ 是 |
| `.env` | 实际环境变量 | ❌ 否（已加入 `.gitignore`） |

### 4.2 `.env.example` 格式要求

1. 使用 `#` 注释说明每个变量的用途和示例值
2. 示例值使用占位符表示敏感信息（如 `your_api_key_here`）
3. 按功能区块分组，用分隔线区分
4. 不包含任何真实密钥

### 4.3 文件模板结构

```bash
# ==========================================
# {功能区块名称}
# {区块说明}
# ==========================================

# {变量说明}
# {示例值列表}
{变量名}={占位符示例值}
```

## 5. 安全规范

### 5.1 敏感信息处理

- API Key 等敏感信息**只能**出现在 `.env` 文件中
- `.env` 必须加入 `.gitignore`
- `.env.example` 中的示例值必须是**明显无效的占位符**

### 5.2 示例值安全标准

| 变量类型 | 示例值格式 | 是否有效 |
|---------|-----------|---------|
| API Key | `your_api_key_here` | ❌ 无效 |
| URL | `https://api.example.com/v1` | ❌ 无效域名 |
| Model | `gpt-4o` | ✅ 有效但公开 |
